# ZK - Community Screen

---

# Purpose

This document defines the structure and user experience of the Community Screen.

The Community Screen is where users discover, create, and interact with meaningful content.

---

# Objectives

Users should be able to:

* Discover inspiring content
* Share experiences
* Learn from others
* Ask questions
* Join discussions
* Build meaningful connections

---

# Screen Structure

Top to Bottom:

* Header
* Search Bar
* Community Stories
* AI Recommendations
* Categories
* Feed
* Trending Posts
* Suggested Communities
* Active Challenges

---

# Header

Display:

* Community title
* Notification button
* Create Post button

---

# Search

Support searching for:

* Posts
* Users
* Communities
* Topics
* Hashtags

---

# Stories

Optional short updates.

Display:

* User avatar
* Progress updates
* Achievements

---

# Categories

Examples:

* Sports
* Health
* Education
* Nutrition
* Volunteering
* Motivation
* Technology
* Lifestyle

---

# Feed

The main content area.

Contains:

* User posts
* Images
* Videos
* Polls
* Progress updates

---

# Trending Section

Display:

* Popular discussions
* Trending hashtags
* Viral educational content

---

# Suggested Communities

Recommend groups based on:

* User interests
* Activity
* AI recommendations

---

# Challenges

Show:

* Active challenges
* Friends participating
* Join button

---

# Floating Action Button

Primary action:

Create New Post

---

# Empty State

If no posts exist:

Display:

* Friendly illustration
* Welcome message
* Create first post button

---

# Design Principles

The Community Screen should always feel:

* Positive
* Modern
* Clean
* Motivating
* Easy to navigate
* Safe
