# ZK - Profile Layout

---

# Purpose

This document defines the layout system of the Profile Screen.

The layout should organize personal information in a clean, modern, and scalable way while keeping important actions easily accessible.

---

# Layout Philosophy

The Profile should feel:

* Personal
* Organized
* Elegant
* Comfortable
* Consistent

---

# Screen Structure

Top to Bottom:

* Safe Area
* Header
* Profile Card
* Statistics
* Achievement Summary
* Activity Timeline
* Favorite Services
* Saved Content
* Quick Actions
* Account Options

---

# Spacing

Horizontal Padding:

24 px

Vertical Section Spacing:

24 px

Card Gap:

16 px

---

# Profile Card

The profile card contains:

* Avatar
* Full Name
* Username
* Bio
* Country
* Join Date

---

# Statistics Layout

Display statistics in a responsive horizontal row.

Each statistic card contains:

* Title
* Value
* Optional Icon

---

# Achievement Section

Display earned badges.

Provide a "View All" button.

---

# Activity Timeline

Display recent activity in chronological order.

Newest activity appears first.

---

# Favorite Services

Display favorite services as horizontal cards.

Allow quick access.

---

# Saved Content

Organize saved items by type:

* Posts
* Articles
* Videos
* Services

---

# Quick Actions

Provide easy access to:

* Edit Profile
* Privacy
* Settings
* Help

---

# Responsive Design

Phones:

Single-column layout.

Tablets:

Adaptive two-column sections where appropriate.

---

# Accessibility

Support:

* Large Text
* Screen Readers
* High Contrast
* Reduced Motion

---

# Performance

Load sections progressively.

Cache frequently viewed profile information.

Maintain smooth scrolling.

---

# Design Principles

The layout should always be:

* Clean
* Balanced
* Readable
* Modern
* Scalable
* Future Ready
