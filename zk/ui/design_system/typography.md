# ZK Design System - Typography

## Philosophy

Typography is one of the strongest elements of the ZK identity.

Text should be clean, readable, modern, elegant, and accessible.

The typography system must create a clear visual hierarchy while maintaining consistency across the platform.

---

# Design Principles

Typography should be:

* Readable
* Elegant
* Consistent
* Responsive
* Accessible
* Minimal
* Professional

Responsive Typography

Font sizes adapt automatically for:

* Phones
* Tablets
* Foldable Devices
* Desktop (Future)

---

Accessibility

Typography must support:

* Dynamic Text Scaling
* Screen Readers
* High Contrast Mode
* Large Font Sizes

Never rely on small text for essential information.

---

General Rules

* Use no more than two font families.
* Maintain clear visual hierarchy.
* Avoid excessive font weights.
* Keep spacing comfortable.
* Prioritize readability over decoration.
