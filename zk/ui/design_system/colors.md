# ZK Design System - Colors

## Philosophy

The ZK color system represents life, growth, energy, trust, and modern technology.

Colors should feel premium, calm, motivating, and accessible.

---

# Primary Colors

## Primary Green

Purpose:

Main brand color.

Used for:

* Primary buttons
* Active elements
* Progress indicators
* Success actions

Suggested Color:

#22C55E

---

## Secondary Blue

Purpose:

* Information

# Accessibility

All colors must provide sufficient contrast.

Never rely on color alone to communicate information.

Support both Light Mode and Dark Mode.

---

# General Rules

Use the Primary Green as the dominant brand color.

Avoid excessive color usage.

Keep interfaces clean and balanced.

Reserve Accent Orange for important highlights only.
