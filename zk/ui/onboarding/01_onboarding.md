# ZK - Onboarding

## Purpose

The onboarding experience introduces new users to the ZK platform, its mission, and its most important features in a simple, inspiring, and engaging way.

The goal is to help users immediately understand the value of ZK and encourage them to begin their journey.

---

# Objectives

* Introduce the ZK vision.
* Present the main features.
* Build trust.
* Create excitement.
* Encourage account creation.

---

# Navigation Logic

If the user completes onboarding:

→ Go to Login.

If the user skips onboarding:

→ Go to Login.

Onboarding is displayed only on the first launch unless reset from Settings.
