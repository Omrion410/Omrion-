# ZK - Home Components

---

# Purpose

This document defines all reusable components used inside the Home Screen.

Every component should be modular, reusable and independently updatable.

---

# Component Principles

Each component must be:

- Independent
- Reusable
- Responsive
- Accessible
- Configurable
- Lightweight

---

# Components List

## 1. Welcome Header

Contains:

- User avatar
- Greeting message
- Username
- Daily motivation
- Notification button

---

## 2. Search Bar

Supports:

- Global search
- Voice search
- AI search
- Recent searches

---

## 3. Daily Goal Card

Displays:

- Today's goal
- Progress percentage
- Completion status
- Quick action button

---

## 4. Quick Actions

Examples:

- Workout
- Nutrition
- AI Coach
- Community
- Challenges

Supports icon and text.

---

## 5. Service Card

Displays:

- Service icon
- Service name
- Short description
- Favorite button
- Badge (optional)

---

## 6. Services Grid

Contains multiple Service Cards.

Supports:

- Grid mode
- List mode
- Dynamic ordering

---

## 7. AI Recommendation Card

Displays:

- Personalized suggestion
- AI explanation
- Action button

---

## 8. Community Card

Displays:

- User avatar
- Username
- Activity preview
- Like count
- Comment count

---

## 9. Challenge Card

Displays:

- Challenge title
- Difficulty
- Progress
- Reward
- Join button

---

## 10. Statistics Card

Displays:

- Steps
- Calories
- Workouts
- Active days
- Streak

---

## 11. News Card

Displays:

- Thumbnail
- Title
- Category
- Publish date

---

## 12. Bottom Navigation

Contains:

- Home
- Services
- Community
- AI
- Profile

---

# Component Rules

All components should support:

- Dark Mode
- Light Mode
- Dynamic Colors
- Responsive Layout
- Smooth Animations
