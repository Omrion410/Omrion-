# ZK - Home Animations

---

# Purpose

This document defines all animations used within the Home Screen.

Animations should improve the user experience without reducing performance.

---

# Animation Principles

Animations must be:

* Smooth
* Fast
* Natural
* Purposeful
* Minimal
* Premium

---

# Screen Entrance

The Home Screen appears using:

* Fade In
* Soft Slide Up

Duration:

350 ms

---

# Header Animation

The Welcome Header fades into view.

User avatar scales slightly.

Greeting text slides upward.

---

# Search Bar

Appears with:

Fade In

Soft Expansion

When selected:

Highlight border

Soft shadow

---

# Service Cards

Cards appear progressively.

Animation:

Fade + Scale

Delay:

40 ms between cards.

---

# Quick Actions

Icons scale gently when pressed.

Scale:

100% → 95% → 100%

Duration:

150 ms

---

# AI Recommendations

Recommendation cards fade in.

Suggested content slides from the right.

---

# Statistics

Numbers animate smoothly from:

0

↓

Current Value

Duration:

700 ms

---

# Notifications

Notification badge:

Small pulse animation.

Only when new notifications exist.

---

# Pull To Refresh

Animation:

Modern circular indicator.

Smooth rotation.

---

# Bottom Navigation

Selected icon:

Scale Up

Color Transition

Duration:

180 ms

---

# Floating Button

Soft floating motion.

Very subtle shadow animation.

---

# Accessibility

If Reduce Motion is enabled:

* Disable floating effects
* Disable pulse
* Keep only Fade transitions

---

# Performance

Maintain:

60 FPS

GPU acceleration.

Avoid heavy animations.

---

# Future Animations

Potential additions:

* Dynamic backgrounds
