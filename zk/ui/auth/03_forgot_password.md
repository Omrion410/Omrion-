# ZK - Forgot Password Screen

## Purpose

The Forgot Password Screen allows users to securely recover access to their ZK account.

---

# Objectives

* Simple password recovery
* Fast verification
* Secure identity confirmation
* Clear user guidance

---

# Layout

Top Section

* ZK Logo
* Forgot Password
* Short explanation

Center Section

* Email Address or Phone Number
* Continue Button

Bottom Section

* Back to Login

---

# Recovery Methods

Users can recover their account using:

* Email Address
* Phone Number

---

# Verification

If Email is selected:

* Send verification code
* Verify email ownership

If Phone Number is selected:

* Send SMS verification code
* Verify phone ownership

---

# Buttons

Primary Button

Continue

Secondary Button

Back to Login

---

# Validation

Validate:

* Email format
* Phone number format

Display validation messages immediately.

---

# Security

* Verification code expires after 5 minutes.
* Limit recovery attempts.
* Prevent brute force attacks.
* Secure communication.

---
