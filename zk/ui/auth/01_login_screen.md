# ZK - Login Screen

## Purpose

The Login Screen allows users to securely access their ZK account while providing a fast, elegant, and trustworthy experience.

---

# Objectives

* Fast authentication
* Secure login
* Simple interface
* Premium user experience

---

# Layout

Top Section

* ZK Logo
* Welcome Back
* Short description

Center Section

* Email or Phone Number
* Password
* Show / Hide Password

Bottom Section

* Forgot Password
* Login Button
* Divider
* Continue with Google
* Continue with Apple
* Continue with Facebook
* Create New Account
