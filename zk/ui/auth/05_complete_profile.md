# ZK - Complete Profile Screen

## Purpose

The Complete Profile Screen allows users to personalize their ZK account after successful verification.

---

# Objectives

* Build a complete user profile
* Personalize the user experience
* Improve recommendations
* Increase community engagement

---

# Layout

Top Section

* Profile Picture
* Welcome Message
* Short Description

Center Section

* Full Name
* Username
* Date of Birth
* Gender
* Country
* City
* Preferred Language

Optional Information

* Occupation
* Education
* Sports Interests
* Personal Goals
* Biography

Bottom Section

* Save Profile Button
* Skip for Now Button

---

# Profile Picture

Allow users to:

* Take a Photo
* Choose from Gallery
* Remove Photo
* Crop Image

Supported Formats:

* JPG
* PNG
* WEBP

---

# Interests

Users may choose multiple interests.

Examples:

* Sports
* Health
* Education
* Volunteering
* Technology

* Travel

* Fitness
* Community

# Goals

Users can select goals such as:

* Build Better Habits
* Grow Professionally
