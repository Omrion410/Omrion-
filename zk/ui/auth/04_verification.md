# ZK - Verification Screen

## Purpose

The Verification Screen confirms the user's identity before allowing access to the account or completing registration.

---

# Objectives

* Secure identity verification
* Fast code validation
* Smooth user experience
* Protect user accounts

---

# Layout

Top Section

* ZK Logo
* Verify Your Identity
* Short explanation

Center Section

* Verification Code Input
* Countdown Timer
* Resend Code Button

Bottom Section

* Verify Button
* Back Button

---

# Verification Methods

Supported methods:

* Email Verification
* SMS Verification

---

# Verification Code

Code Length:

6 Digits

Input Style:

Separate boxes

Auto Focus:

Enabled

Auto Paste:

Supported

---

# Timer

Default:

60 Seconds

When timer ends:

Enable "Resend Code"

---

# Buttons

Primary Button

Verify

Secondary Button

Resend Code

Text Button

Back

---

# Validation

Check:

* Code length
* Correct code
* Expiration time

Display clear error messages when verification fails.

---

# Security

* Code expires after 5 minutes.
* Maximum verification attempts: 5
* New code generated after resend.
* Encrypted verification process.
