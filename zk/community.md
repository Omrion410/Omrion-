# Community System

## Overview

The Community System is the social hub of the Zk platform. It enables users to connect, share knowledge, exchange experiences, inspire others, and build meaningful relationships around education, careers, health, sports, volunteering, travel, and personal development.

---

# Objectives

* Build an active and positive community.
* Encourage knowledge sharing.
* Inspire personal growth.
* Promote collaboration.
* Increase user engagement.
* Support networking opportunities.

---

# Community Features

Users can:

* Create posts.
* Edit posts.
* Delete posts.
* Like posts.
* Comment on posts.
* Reply to comments.
* Share posts.
* Save posts.
* Report inappropriate content.

---

# Post Types

Supported content includes:

* Text Posts
* Images
* Videos
* Documents
* Polls
* Questions
* Achievement Stories
* Opportunity Sharing
